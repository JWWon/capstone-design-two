# Installation

1. Create conda env

   ```bash
   ~$ conda env create --name {any_names_you_want} --file environment.yaml
   ```

2. Activate conda env

   ```bash
   ~$ conda activate {any_names_you_want}
   ```

3. Install some packages manually

   ```bash
   ~$ pip install pygame
   ```
